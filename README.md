# 📊 Mobile Sales Analysis Dashboard

## 📌 Overview

This project analyzes mobile sales data using Power BI to track performance, trends, and customer behavior.

---

## 📷 Dashboard Preview

![Mobile Sales Dashboard](https://github.com/kanchanawari12/mobile_sales_analysis/blob/main/Screenshot%202026-04-06%20224349.png)

---

## 🔄 Data Preparation

* Cleaned data (removed nulls & duplicates)
* Changed data types
* Created a calendar table for date analysis
* Built relationship between tables

---

## 📈 Dashboard Features

* KPIs: Total Sales, Quantity, Average Price
* Sales by City and Mobile Model
* Payment Method Distribution
* Sales Trend over Time
* Interactive slicers

---

## 💡 Insights

* Identified top-performing cities and products
* Observed monthly trends
* Analyzed customer behavior

---

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX

---

## 🚀 Outcome

Interactive dashboard for data-driven decisions.
